# 会务管理系统 · 用户操作手册

> 本文面向初次使用本系统的会务组织者/管理员，采用“按模块教学”的方式，帮助您在最短时间内掌握后台各项功能。

---

## 1. 系统简介

本系统是一个面向会议/活动的“一站式会务管理平台”，主要包含两大部分：

- **前台站点**：面向参会者，展示会议信息、报名、投稿、日程、讲者、酒店、图片直播等。
- **管理后台**：面向组织者，用于维护会议内容、审核报名/投稿/预订、管理嘉宾、签到、导出数据等。

所有前台内容均通过后台维护，无需编写代码。

---

## 2. 登录后台

1. 在浏览器中打开系统地址，例如：`http://localhost:3003/admin`（实际地址以部署方提供为准）。
2. 使用管理员账号登录。系统初始化后会默认生成一个管理员账号：
   - **邮箱**：`admin@conf.local`
   - **密码**：`admin123`
3. 首次登录后，建议立即通过「用户管理」修改默认管理员密码，或创建新的管理员账号并停用默认账号。

> **安全提示**：`admin@conf.local` 仅为初始账号，正式上线前务必改密或删除。

---

登录成功后，您会看到后台左侧有一列导航菜单，右侧为内容区。

| 菜单 | 功能 |

| 首页宫格 | 配置前台首页的功能入口卡片 |
| 直播管理 | 配置多个直播会场的外部链接 |
| 通知管理 | 发布会议通知公告 |
| 内容页 | 编辑活动简介、会场交通、联系方式等富文本页面 |
| 日程管理 | 创建会议日程场次，关联讲者 |
| 讲者管理 | 维护讲者名单、上传照片、发送邀约认证链接 |
| 报名管理 | 审核报名、设置参会类型、导出名单 |
| 酒店管理 | 维护可预订酒店信息 |
| 预订管理 | 审核酒店预订申请 |
| 投稿审核 | 审核论文投稿，下载附件 |
| 直播图片 | 创建相册、上传现场照片 |

| 胸卡模板 | 配置胸卡样式，批量导出 PDF |
| 用户管理 | 管理系统用户、重置密码、查看跨会议报名记录 |
| 签到管理 | 扫码签到、手动签到、查看签到统计 |

### 3.2 重要概念：当前会议

本系统支持管理多个会议。进入后台后，必须先选择或创建一个“当前会议”，否则大部分菜单会被禁用。具体操作见下一章「会议管理」。

## 4. 会议管理（第一步必须先做）

所有会议数据都是隔离的。在使用其他功能前，请先进入 **「会议管理」**。

### 4.1 创建会议

1. 点击左侧菜单「会议管理」。
2. 点击右上角 **「+ 新建会议」**。
3. 填写会议信息：
   - **会议名称***：必填，例如“2025 人工智能学术年会”。
   - **地点**：会议举办地点。
   - **开始日期 / 结束日期**：会议日期。
   - **简介**：会议描述，可选。
   - **报名后需要审核**：勾选后，用户报名仍需管理员手动审核（当前版本默认均为待审核）。
   - **开放报名 / 截止报名**：报名时间段，可选。
   - **人数上限**：报名最大人数，0 或不填表示不限制。
4. 点击 **「保存」**。

> 第一个创建的会议会自动设为“默认会议”，即用户访问首页时看到的会议。

### 4.2 选择/切换当前会议

1. 在「会议管理」列表中，找到目标会议。
2. 点击该行右侧的 **「进入管理」** 按钮。
3. 系统会将该会议设为当前管理会议，左侧菜单自动启用。

### 4.3 设为默认会议

默认会议是用户访问首页 `/` 时自动展示的会议。点击会议行右侧的 **「设为默认」** 按钮即可切换。

### 4.4 编辑会议

点击会议行右侧的 **「编辑」** 按钮，可修改会议名称、日期、地点、简介、报名时间等。

### 4.5 删除会议

点击会议行右侧的 **「删除」** 按钮。删除会议会**级联删除**该会议下的所有数据（报名、投稿、日程、酒店、嘉宾等），请谨慎操作。

### 4.6 分享报名链接

在会议列表中，每行都有一个「报名链接」复制按钮，点击即可复制该会议的专属报名链接，例如：

```
http://localhost:3003/r/xxxxxxxxx
```

可将此链接放入邀请函、公众号、邮件中。

## 5. 基础信息（仪表盘）

选择会议后，点击左侧「基础信息」，进入当前会议的仪表盘。

### 5.1 数据概览

页面顶部有 6 个统计卡片，实时显示：

- **报名**：当前会议报名总人数。
- **已签到**：已完成签到的人数。
- **预订**：酒店预订申请数。
- **投稿**：论文投稿数。
- **嘉宾**：嘉宾人数。
- **渠道**：推广渠道数量。

### 5.2 站点设置

在「站点设置」区域，可以配置前台展示内容：

| 字段 | 说明 |
|---|---|
| 会议名称 | 前台页面标题 |
| 会议日期 | 显示在首页的日期 |
| 会议地点 | 显示在前台的地点 |
| Logo | 网站顶部 Logo 图片 |
| 主图 | 首页顶部横幅背景图 |
| 直播地址 | 前台直播入口跳转的外部链接 |
| 欢迎致辞 | 首页显示的欢迎文字（支持富文本） |
| 页脚内容 | 网站底部信息（支持富文本） |
| 会场地址 | 地图定位地址 |
| 经度 / 纬度 | 高德地图坐标，用于会场地图展示 |

修改后点击页面中的保存按钮即可生效。

### 5.3 数据导出

同一页面提供以下快捷导出按钮：

- 报名名单 CSV
- 酒店预订 CSV
- 签到记录 CSV
- 渠道统计 CSV

点击后浏览器会自动下载对应文件，可用 Excel 打开。

## 6. 首页宫格设计

首页宫格是前台首页的“功能入口卡片”，例如“会议日程”“讲者介绍”“酒店预订”等。

### 6.1 添加入口

1. 点击左侧「首页宫格」。
2. 填写标题、链接（如 `/schedule`）、选择图标。
3. 选择卡片尺寸：
   - **SMALL**：1×1 小卡片
   - **WIDE**：2×1 横向卡片
   - **TALL**：1×2 纵向卡片
   - **LARGE**：2×2 大卡片
4. 上传背景图（可选）。
5. 点击保存。

### 6.2 调整与隐藏

- 拖拽可调整入口顺序。
- 点击隐藏按钮，该入口不会在前台显示。
- 右侧通常有实时预览，可直观看到前台效果。

> 背景图支持直接上传到阿里云 OSS，上传后会自动回填图片地址。

## 7. 直播管理

如果会议有多个线上直播会场，可在此配置。

1. 点击左侧「直播管理」。
2. 点击添加直播。
3. 填写：
   - **名称**：如“主会场直播”“分论坛 A”。
   - **链接**：外部直播地址（如腾讯会议、B站、视频号等）。
   - **封面图**：可选。
   - **描述**：可选。
   - **时间**：场次时间，可选。
4. 保存后，前台直播页面会列出所有入口，点击后在新标签页打开外部链接。

---

## 8. 通知管理

用于发布会议通知、公告、提醒。

### 8.1 新建通知

1. 点击左侧「通知管理」。
2. 在“新建通知”区域填写：
   - **标题**：通知标题。
   - **正文**：通知内容，纯文本即可，换行会自动分段。
   - **立即发布**：默认勾选，取消勾选则为草稿，前台不显示。
3. 点击 **「新建」**。

### 8.2 编辑/删除通知

- 在列表中点击 **「编辑」** 可修改通知标题、正文、发布状态。
- 点击 **「删除」** 可删除通知。

## 9. 内容页管理

内容页是前台固定的富文本页面，包括：

- 活动简介
- 活动说明
- 活动须知
- 会场交通
- 联系方式

### 9.1 编辑内容页

1. 点击左侧「内容页」。
2. 找到需要编辑的页面，点击 **「编辑」** 或 **「创建」**。
3. 填写页面标题和正文内容（支持富文本，可插入图片、链接等）。
4. 保存后，前台对应菜单会显示该内容。

---

## 10. 日程管理

用于创建会议日程表，并与讲者关联。

### 10.1 新建场次

1. 点击左侧「日程管理」。
2. 在顶部表单填写：
   - **日期**：如 2025-08-20。
   - **开始时间 / 结束时间**：如 09:00 / 10:00。
   - **会场**：如“主会场 A”。
   - **场次标题**：如“开幕式”。
   - **讲者**：选择已创建的讲者（可多选）。
3. 点击 **「新建场次」**。

> 如果讲者列表为空，请先去「讲者管理」创建讲者。

### 10.2 编辑/删除场次

- 点击列表中的 **「编辑」** 可修改场次信息。
- 点击 **「删除」** 可删除该场次。

前台日程页面会按“日期 → 会场 → 开始时间”自动分组展示。

---

## 11. 讲者管理

用于维护讲者/主持人信息，并与日程关联。

### 11.1 新建讲者

1. 点击左侧「讲者管理」。
2. 在顶部表单填写：
   - **姓名***：必填。
   - **职称**：如教授、博士。
   - **单位**：所属机构。
   - **讲者照片**：上传头像或照片。
   - **简介**：支持 HTML 富文本。
3. 点击 **「新建」**。

### 11.2 编辑讲者

点击列表中的 **「编辑」** 按钮，可修改讲者资料、照片、简介。

### 11.3 邀约认证

如果讲者需要登录系统上传自己的报告资料，可发送邀约认证链接：

1. 在讲者列表中，点击该讲者右侧的 **「邀约认证」** 按钮。
2. 系统生成一个专属链接，例如 `/s/xxxxxxxxx`。
3. 将链接复制给讲者。
4. 讲者打开链接并登录后，点击“接受邀约”，其账号即与该讲者记录绑定。
5. 绑定后，讲者可在个人中心看到“上传报告资料”入口。

### 11.4 删除讲者

点击 **「删除」** 按钮可删除讲者。删除讲者会同时删除其关联的日程关系和上传资料记录。

---

## 12. 报名管理

用于审核参会报名、设置参会类型、导出名单。

### 12.1 查看报名列表

点击左侧「报名管理」，列表显示当前会议的所有报名信息，包括：

- 姓名
- 邮箱
- 参会类型
- 单位
- 状态（待审核 / 已通过 / 已拒绝）
- 签到状态

### 12.2 审核报名

- 找到需要审核的报名记录。
- 点击该行右侧的 **「通过」** 或 **「拒绝」** 按钮。
- 状态会立即更新。

### 12.3 设置参会类型

页面下方为“参会类型设置”区域：

1. 点击“添加类型”。
2. 填写类型名称（如“普通参会”“学生票”“VIP”）、费用、描述。
3. 保存后，前台报名表单会同步显示这些类型。

> 删除已有报名的类型时，系统会提示将相关报名批量转移到其他类型。

### 12.4 导出报名名单

点击页面右上角的 **「导出 CSV」** 按钮，可下载 Excel 文件，包含所有报名记录。

## 13. 酒店管理

用于维护会议合作的酒店信息，供前台用户预订。

### 13.1 新建酒店

1. 点击左侧「酒店管理」。
2. 在顶部表单填写：
   - **名称**：酒店名称。
   - **价格/晚**：参考价格。
   - **地址**：酒店地址。
   - **距离**：距离会场的距离。
   - **酒店图片**：上传酒店图片。
   - **简介**：酒店介绍。
3. 点击 **「新建酒店」**。

### 13.2 编辑/删除酒店

- 点击 **「编辑」** 修改酒店信息。
- 点击 **「删除」** 删除酒店。如果该酒店已有预订记录，则无法删除。

---

## 14. 预订管理

用于审核用户提交的酒店预订申请。

1. 点击左侧「预订管理」。
2. 列表显示所有预订申请，包括：酒店名称、入住/离店日期、房间数、状态。
3. 点击 **「通过」** 或 **「拒绝」** 进行审核。
4. 点击 **「导出 CSV」** 可下载预订明细。

---

## 15. 投稿审核

用于审核用户提交的论文/报告投稿。

1. 点击左侧「投稿审核」。
2. 列表显示投稿标题、作者、提交人、状态。
3. 点击 **「通过」** 或 **「拒绝」** 进行审核。
4. 如果投稿包含 PDF 附件，可点击下载。
5. 点击 **「导出 CSV」** 可下载投稿列表。

## 16. 直播图片（相册管理）

用于上传会议现场照片，形成图片直播效果。

### 16.1 创建相册

1. 点击左侧「直播图片」。
2. 在“新建相册”区域填写标题、选择日期。
3. 点击 **「新建相册」**。

### 16.2 上传照片

1. 在相册卡片中找到“上传照片”入口。
2. 选择图片文件（支持 jpeg/png/webp，单张 ≤5MB）。
3. 上传后，照片会显示在相册中，前台用户可实时查看。

### 16.3 删除照片/相册

- 点击照片下方的 **「删除」** 按钮删除单张照片。
- 点击相册标题右侧的 **「删除」** 按钮删除整个相册（会同时删除相册内所有照片）。

## 17. 嘉宾管理

用于维护 VIP、嘉宾、媒体等特殊参会人员信息，并发送邀请函。

### 17.1 新建嘉宾

1. 点击左侧「嘉宾管理」。
2. 点击右上角 **「+ 新建嘉宾」**。
3. 填写姓名、单位、联系方式、级别（VIP / 嘉宾 / 媒体）等信息。
4. 保存。

### 17.2 批量导入嘉宾

如果嘉宾较多，可点击 **「导入嘉宾」** 按钮，按系统要求的 CSV 格式批量导入。

### 17.3 发送邀请函

1. 在嘉宾列表中，点击该嘉宾右侧的 **「复制邀请链接」** 按钮。
2. 系统生成专属链接，例如 `/g/xxxxxxxxx`。
3. 将链接通过微信/邮件发送给嘉宾。
4. 嘉宾打开链接可查看会议信息、个人接待安排，并点击“确认出席”。

### 17.4 筛选与搜索

- 页面上方可按级别筛选：全部 / VIP / 嘉宾 / 媒体。
- 搜索框可输入姓名快速查找。

---

## 18. 接待管理

用于维护嘉宾和报名者的接待安排，包括交通、酒店、车辆、房间号等信息。

### 18.1 查看接待总览

点击左侧「接待管理」，页面显示一个总览表格，包含：

- 来源（嘉宾 / 报名）
- 姓名、单位、分类
- 联系方式
- 抵达信息（航班/高铁/自驾、车次/航班号、时间、地点）
- 返程信息
- 酒店名称、房间号
- 司机/车牌

### 18.2 编辑接待信息

1. 找到需要编辑的人员。
2. 点击 **「编辑」** 按钮。
3. 填写或修改交通、酒店、车辆、备注等信息。
4. 保存。

> 嘉宾的接待信息在「嘉宾管理」中编辑，报名者的接待信息在「报名管理」中点击姓名进入编辑。

### 18.3 快速修改房间号

在接待总览表格中，可直接在“房间号”单元格内输入房间号并保存，无需进入编辑页。

### 18.4 导出接待信息

点击页面右上角的 **「导出」** 按钮，可下载接待安排明细表。

## 19. 渠道推广

用于创建推广渠道，统计不同渠道的访问量、报名量、签到转化率。

### 19.1 新建渠道

1. 点击左侧「渠道推广」。
2. 点击右上角 **「+ 新建渠道」**。
3. 填写：
   - **渠道名称**：如“微信公众号”“合作院校”。
   - **短码**：用于标识渠道，如 `wechat`、`school-a`。
   - **负责人**：可选。
4. 保存。

### 19.2 分发渠道链接

渠道创建后，系统会生成专属链接，例如：

```
http://localhost:3003/register-conf?ch=wechat
```

将该链接发布到对应渠道，用户通过此链接访问、报名，数据会自动归属到该渠道。

### 19.3 查看统计数据

在渠道列表中，可查看每个渠道的：

- **PV**：页面访问量。
- **UV**：独立访客数。
- **报名**：通过该渠道报名的人数。
- **签到**：通过该渠道报名且已签到的人数。
- **转化率**：报名 / UV。

### 19.4 导出渠道统计

点击 **「导出 CSV」** 可下载渠道统计表。

## 20. 胸卡模板

用于配置参会胸卡的样式，并批量导出 PDF。

### 20.1 配置模板

1. 点击左侧「胸卡模板」。
2. 设置胸卡尺寸（默认 86mm × 147mm）。
3. 上传背景图（可选）。
4. 调整各字段在胸卡上的位置：
   - 姓名
   - 职务/职称
   - 单位
   - 二维码（签到凭证）
   - 会议标题
5. 设置各字段的 X/Y 坐标和字号。
6. 点击 **「预览」** 查看效果。

### 20.2 导出胸卡

确认样式无误后，点击 **「导出所有报名者胸卡」** 按钮，系统会生成一个 PDF 文件，包含所有非“已拒绝”状态的报名者胸卡。

> 二维码指向 `/c/{token}`，现场扫码可快速完成签到。

## 22. 签到管理

用于现场签到，支持扫码签到和手动签到两种方式。

### 22.1 查看签到统计

点击左侧「签到管理」，页面顶部显示：

- 报名总人数
- 已签到人数
- 未签到人数

下方显示最近签到记录。

### 22.2 扫码签到

1. 点击 **「扫码签到」** 卡片。
2. 允许浏览器使用摄像头。
3. 对准参会者个人中心的二维码进行扫描。
4. 扫描成功后，系统会显示参会者信息并自动完成签到。

### 22.3 手动签到

1. 点击 **「手动签到」** 卡片。
2. 输入参会者姓名、手机号、邮箱或单位进行搜索。
3. 找到目标记录后，点击 **「签到」** 按钮完成签到。

### 22.4 导出签到记录

在「基础信息」页面中，点击 **「签到记录 CSV」** 可导出全部签到明细，包括签到时间、签到方式（扫码/手动/自助）。

## 24. 常见问题

### Q1：为什么进入后台后大部分菜单是灰色的？

A：您还没有选择“当前会议”。请先点击「会议管理」，选择或创建一个会议，然后点击「进入管理」。

### Q2：用户报名后状态一直是“待审核”？

A：当前版本所有报名创建后默认均为“待审核”，需要管理员在「报名管理」中手动点击“通过”。

### Q3：如何让用户看到最新发布的通知？

A：在「通知管理」中新建通知时，确保勾选“立即发布”。未发布的通知前台不会显示。

### Q4：删除会议后数据还能恢复吗？

A：不能。删除会议会级联删除该会议下的所有数据，请提前导出备份。

### Q5：讲者如何上传自己的 PPT？

A：管理员在「讲者管理」中点击“邀约认证”，将生成的链接发送给讲者。讲者接受邀约后，在个人中心即可上传报告资料。

### Q6：如何查看某个渠道推广效果？

A：在「渠道推广」中，列表会显示每个渠道的 PV、UV、报名、签到、转化率数据，也可导出 CSV。

### Q7：上传的图片/文件为什么访问不到？

A：请确认 `public/uploads/` 目录存在且程序有写入权限。生产环境多实例部署时，建议使用对象存储（需联系技术部署人员配置）。

### Q8：如何修改网站 Logo 和首页大图？

A：在「基础信息」→「站点设置」中上传 Logo 和主图，保存后前台自动更新。

## 25. 快速上手清单

首次使用本系统时，建议按以下顺序操作：

1. **登录后台**，修改默认管理员密码。
2. **创建会议**：填写会议名称、日期、地点。
3. **进入会议管理**：点击「进入管理」，将新建的会议设为当前会议。
4. **配置站点**：在「基础信息」中上传 Logo、主图，填写欢迎致辞、页脚、联系方式。
5. **设计首页**：在「首页宫格」中添加日程、讲者、酒店、报名等入口。
6. **发布内容**：在「内容页」中编辑活动简介、会场交通、联系方式。
7. **维护数据**：
   - 在「酒店管理」中添加合作酒店。
   - 在「讲者管理」中添加讲者并发送邀约认证。
   - 在「日程管理」中创建日程并关联讲者。
   - 在「报名管理」中设置参会类型。
8. **开放报名**：复制「会议管理」中的报名链接，分享给潜在参会者。
9. **审核报名/投稿/预订**：用户提交后，及时在后台审核。
10. **现场执行**：使用「签到管理」扫码签到，使用「接待管理」查看接待安排，使用「胸卡模板」导出胸卡。
